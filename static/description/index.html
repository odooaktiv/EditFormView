Open Edit Form View

The primary purpose of this module is to open default edit form view when clicked list view and kanban view.

Below listed are features of Open Edit Form View:

  • User open default edit form view when clicked list view and kanban view in Odoo.

Prerequisites before Installation

There are no prerequisites required before the Installation

When clicked list view record "SO007".

After click on the list view record redirected to open form view default as edit mode not readonly mode.

When clicked Kanban view record "SO007".

After click on the list view record redirected to open form view default as edit mode not readonly mode.
